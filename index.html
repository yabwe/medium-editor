MediumEditor

The dead simple inline editor toolbar.

Try highlighting this text.

Or check the full demo.

  • Pure javascript.
    No jQuery!
  • Lightweight:
    ~28kB gzipped.
  • Chrome, Firefox,
    Safari, IE, & Edge support

Installation

Via npm

npm install medium-editor

Via bower

bower install medium-editor

CDN / jsdelivr

<script src="//cdn.jsdelivr.net/medium-editor/latest/js/medium-editor.min.js"></script>
<link rel="stylesheet" href="//cdn.jsdelivr.net/medium-editor/latest/css/medium-editor.min.css" type="text/css" media="screen" charset="utf-8">

Self-hosted · Download files ⇓

<script src="js/medium-editor.js"></script>
<link rel="stylesheet" href="css/medium-editor.css">
<link rel="stylesheet" href="css/themes/default.css">

Usage

<div class="editable"></div>
<script>var editor = new MediumEditor('.editable');</script>

Extensions

Image Upload

by Pavel Linkesch

Pavel Linkesch has developed a jQuery plugin to upload images & embed content (from Twitter, Youtube, Vimeo, etc.) following Medium.com functionality. Check it out at orthes.github.io/medium-editor-insert-plugin.

Markdown

by Ionică Bizău

Ionică Bizău has created a Medium Editor extension, named Medium Editor Markdown, to add the functionality to render the HTML into Markdown code. Check it out at github.com/IonicaBizau/medium-editor-markdown.

MultiPlaceholders

by Kasi Chonpimai

Kasi Chonpimai has created an enhanced placeholder plugin which enables users to add multiple placeholders with specific HTML tags. Check it out here: github.com/smiled0g/medium-editor-multi-placeholders-plugin

Tables

by yabwe

A tables support extension is available at github.com/yabwe/medium-editor-tables.

Inserting custom HTML

by Ionică Bizău

At @jillix, Ionică Bizău developed a Medium Editor extension that allows inserting custom HTML into the editor. Check it out here: github.com/jillix/medium-editor-custom-html.

MediumButton

by Patrick Stillhart

Patrick Stillhart developed a new and improved way to add buttons to our toolbar. With it you can add HTML or javascript buttons. Check it out at: github.com/arcs-/MediumButton

TCMention for @username or #tags

by Tom Chen

Tom Chen has built this extension to add mention support that came from Medium 2.0. It also allows users to customize the mention panel for their favor. It's available on npm and an UMD version as well. Check out the demo and source code at: github.com/tomchentw/medium-editor-tc-mention
